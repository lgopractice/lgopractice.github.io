    Remove Outermost Parentheses

    Given a valid parentheses string s, return it after removing outermost parentheses from each substring.

    Expand

    If you look at the provided examples, all we need to do is strip the outermost (). To do so, we essentially need to drop all parentheses at depth 0.

    Let's start simple with the following boilerplate code. We process the string character by character, keeping track of the depth. Every time we encounter an opening parentheses, that's increasing depth, and every time we encounter a closing parentheses, that's a decreasing depth.

    fun removeOuterParentheses(s: String): String {
  val sb = StringBuilder()
  val stack = java.util.ArrayDeque<Char>()
  var depth = 0

  for (c in s) {
    if (c == '(') {
      if (depth > 0) sb.append(c)

      stack.push(c)
      depth++
    } else {
      stack.pop()
      depth--

      if (depth > 0) sb.append(c)
    }
  }
  return sb.toString()
}

    Why is the sb.append placement not symmetrical? We can explain that by an example

    Cursor position Illustration Evaluation of depth
    \(0\) (())
    ^•••    		 depth > 0 check should happen first
    \(1\) (())
    •^••    		 depth is \(1\) so append it to sb
    \(2\) (())
    ••^•    		 depth > 0 check should happen later
    \(3\) (())
    •••^    		 depth is \(0\) so don't append it to sb
