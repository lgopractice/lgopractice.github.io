    Final Prices With a Special Discount in a Shop

    A shop has \(n\) items laid out in a sequence. A discount is in effect, where if you buy \(i^{th}\) item, you will receive a discount equivalent of \(\text{price}_j\)

    j is the first item on \(i\)'s right (\(i < j\)) such that \(\text{price}_j \le \text{price}_i\)

    Example

    • Input: price = [8, 4, 6, 2, 3]
    • Output: [4, 2, 4, 2, 3]
    • Discount of 4 on 8.
    • Discount of 2 on 4.
    • Discount of 2 on 6.
    • No discount on 2.
    • No discount on 3.
    • Input: price = [1, 2, 3, 4, 5]
    • Output: [1, 2, 3, 4, 5]
    • Input: price = [10, 1, 1, 6]
    • Output: [9, 0, 1, 6]

    Solution

    Starting from right to left, we use a monotonic stack on \((i, n)\). Pop elememnts as long as \(top > n\). We'll be left with either an empty stack (no discount), or a \(top \le n\), which is out discount.

    fun finalPrices(prices: IntArray): IntArray {
  val stack = ArrayDeque<Int>()
  for ((i, p) in prices.withIndex().reversed()) {
    while (stack.isNotEmpty() && stack.last() > p) {
      stack.removeLast()
    }
    if (stack.isNotEmpty()) {
      prices[i] = p - stack.last()
    }
    stack.addLast(p)
  }
  return prices
}
    • \(8, 4, 6, 2, \fbox{3}\) stack has nothing, so no discount.
    • \(8, 4, 6, \fbox{2}, 3\) stack is \((3]\), and we pop it as long as \(top > 2\). Nothing is left on stack, so no discount.
    • \(8, 4, \fbox{6}, 2, 3\) stack is \((2]\), so our discount is \(2\), leading to \(6-2\) price.
    • \(8, \fbox{4}, 6, 2, 3\) stack is \((6, 2]\). We pop \(6\) and end up with \(2\) discount.
    • \(\fbox{8}, 4, 6, 2, 3\) stack is \((4, 2]\). We get a \(4\) discount.
