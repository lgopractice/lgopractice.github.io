# Implement Stack using Queues

Implement a stack using only two queues. As a follow-up, see if you can achieve the same with just a single queue.

```kotlin
class MyStack() {
  fun push(x: Int) {
    // Fill here
  }

  fun pop(): Int {
    // Fill here
  }

  fun top(): Int {
    // Fill here
  }

  fun empty(): Boolean {
    // Fill here
  }
}
```

## Solution

??? "Expand"

    We use a single underlying queue $q$.

    - `push` on `MyStack` translates to `q.enqueue`, followed by $n-1$ `dequeue`-`enqueue`.
    - `pop` on `MyStack` translates to `q.dequeue` where we treat the front of $q$ as the top of the stack.

    $$
    \begin{align}
    \text{stack} &= (] \\
    \text{push}(1) &= (1] \\
    \text{push}(2) &= (1, 2] \rightarrow (2, 1] \\
    \text{push}(3) &= (2, 1, 3] \rightarrow (3, 2, 1] \\
    \end{align}
    $$

    where $($ is the front of the queue.

    ```kotlin
    class MyStack() {

      val queue = java.util.ArrayDeque<Int>()

      fun push(x: Int) {
        queue.add(x)
        repeat(queue.size - 1) {
          queue.add(queue.removeFirst())
        }
      }

      fun pop(): Int = queue.removeFirst()

      fun top(): Int = queue.first()

      fun empty(): Boolean = queue.isEmpty()
    }
    ```
