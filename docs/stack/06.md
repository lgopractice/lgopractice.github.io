# Backspace String Compare

Given two strings $s$ and $t$, check if they are equal when typed in an empty text editor. Treat `#` as a backspace character. `#` on empty string will give empty string.

```kotlin
fun backspaceCompare(s: String, t: String): Boolean {
  // Fill here
}
```

## Example

- `ab#c` and `ad#c` are the same `"ac"`.
- `ab##` and `c#d#` are the same `""`. 
- `a##c` and `"#a#c"` are the same `"c"`.

## Solution

??? "What will not work"

    Processing both strings left to right and skipping `#` will obviously not work. Once we have $i=1$ and $j=1$:

    $$
    \begin{align}
    s &= a\fbox{b}\#c \\ 
    t &= a\fbox{d}\#c
    \end{align}
    $$

    our checks will fail too early.

    ----------

    An obvious plan comes to mind where we process both strings in reverse instead.

    ```kotlin
    fun backspaceCompare1(s: String, t: String): Boolean {
      var i = s.length - 1
      var j = t.length - 1
      while (i >= 0 && j >= 0) {
        if (s[i] == '#') {
          i -= 2
          continue
        }
        if (t[j] == '#') {
          j -= 2
          continue
        }
        if (s[i] != t[j]) return false
        i--
        j--
      }
      return i == -1 && j == -1
    }
    ```

    However, this fails for strings like `"a##c"`, as we jump from the rightmost `#` to `a` and end up not deleting it.

??? "Expand"

    We compare not the strings themselves, but the "derived" string.

    ```kotlin
    fun backspaceCompare(s: String, t: String): Boolean {
      return s.derivedString().compare(t.derivedString())
    }
    ```

    Here "derived" string strips `#` and its left neighbour.

    ```kotlin
    private fun String.derivedString(): java.util.ArrayDeque<Char> {
      val stack = java.util.ArrayDeque<Char>()
      for (c in this) {
        if (c != '#') stack.push(c)
        else {
          if (stack.isNotEmpty()) stack.pop() // (1)
        }
      }
      return stack
    }
    ``` 

    1. repeated backspace in an empty text editor are allowed. we are just left with empty string.

    where `compare` is as follows:

    ```kotlin
    private fun java.util.ArrayDeque<Char>.compare(other: java.util.ArrayDeque<Char>): Boolean {
      while (isNotEmpty() && other.isNotEmpty()) {
        if (pop() != other.pop()) return false
      }
      return isEmpty() && other.isEmpty()
    }
    ```
