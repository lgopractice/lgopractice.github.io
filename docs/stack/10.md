# Daily Temperatures

Given a list of `temperatures`, return `answer` where `answer[i]` is the _number of days you have to wait to get a warmer temperature_.

```kotlin
fun dailyTemperatures(temperatures: IntArray): IntArray {
  // Fill here
}
```

## Example

=== "First"

    - Input = `30, 60, 90`
    - Output = `1, 1, 0`

=== "Second"

    - Input = `73, 74, 75, 71, 69, 72, 76, 73`
    - Output = `1, 1, 4, 2, 1, 1, 0, 0`


## Solution

??? "Expand"

    Not much to this problem. It's a typical monotonic stack solution, except we are putting in indices in the stack and not the actual values.

    ```kotlin
    fun dailyTemperatures(temperatures: IntArray): IntArray {
      val stack = java.util.ArrayDeque<Int>() // indices
      val result = IntArray(temperatures.size)
      for ((i, t) in temperatures.withIndex().reversed()) {
        while (stack.isNotEmpty() && temperatures[stack.peek()] <= t) {
          stack.pop()
        }
        result[i] = if (stack.isEmpty()) 0 else stack.peek() - i
        stack.push(i)
      }
      return result
    }
    ```
