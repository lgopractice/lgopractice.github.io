# Next Greater Node in Linked List

Given a linked list, find the value of **next greater node**.

=== "Placeholder"

    ```kotlin
    fun nextLargerNodes(head: ListNode?): IntArray {
      // Fill here
    }
    ```

=== "`ListNode` definition"

    ```kotlin
    class ListNode(var `val`: Int) {
      var next: ListNode? = null

      constructor(`val`: Int, next: ListNode): this(`val`) {
        this.next = next
      }

      override fun toString(): String {
        val sb = StringBuilder()
        var cursor: ListNode? = this
        while (cursor != null) {
          sb.append(cursor.`val`)
          cursor = cursor.next
          if (cursor != null) {
            sb.append(" -> ")
          }
        }
        return sb.toString()
      }
    }
    ```

## Solution

??? "Expand"

    Obviously we can use a monotonic stack, but the issue is that this linked list implementation does not allow right-to-left traversal.

    To handle this, we reverse the list.

    ```kotlin title="Reverse list and at the same time find its length"
    private fun reverse(head: ListNode?): Pair<ListNode?, Int> {
      var size = 0
      var curr = head
      var prev: ListNode? = null
      while (curr != null) {
        val next = curr.next
        curr.next = prev

        prev = curr
        curr = next
        size++
      }
      return prev to size
    }
    ```

    With this, monotonic stack can now be applied in a straighforward manner.

    ```kotlin
    fun nextLargerNodes(head: ListNode?): IntArray {
      var (head, size) = reverse(head)
      val stack = java.util.ArrayDeque<Int>()
      val answer = IntArray(size)

      var i = size - 1
      while (head != null) {
        while (stack.isNotEmpty() && stack.peek() <= head.`val`) {
          stack.pop()
        }
        answer[i--] = if (stack.isEmpty()) 0 else stack.peek()
        stack.push(head.`val`)
        head = head.next
      }
      return answer
    }
    ```
