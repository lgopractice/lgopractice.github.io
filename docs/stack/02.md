# Valid Parentheses

Given a string containing just `(){}[]`, determine if it's valid.

```kotlin
fun isValid(s: String): Boolean {
  // Fill here
}
```

## Solution

??? "First approach"

    ```kotlin
    fun isValid(s: String): Boolean {
      val stack = java.util.ArrayDeque<Char>()
      for (c in s) {
        when(c) {
          '(', '{', '[' -> stack.push(c)
          else -> {
            if (stack.isEmpty()) return false
            if (stack.pop() != c.opener) return false
          }
        }
      }
      return stack.isEmpty()
    }

    private val Char.opener: Char
      get() = when(this) {
        ')' -> '('
        '}' -> '{'
        ']' -> '['
        else -> '\n'
      }
    ```
