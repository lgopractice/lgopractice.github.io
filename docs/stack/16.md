# Maximum Width Ramp


## Description

A ramp in an integer array nums is a pair `(i, j)` for which $i < j$ and `nums[i] <= nums[j]`. The width of such a ramp is $j - i$.

Given an integer array `nums`, return the maximum width of a ramp in `nums`. If there is no ramp in `nums`, return `0`.

## Examples

=== "Example 1"

    - Input: nums = `[6, 0, 8, 2, 1, 5]`
    - Output: `4`
    - Explanation: The maximum width ramp is achieved at `(i, j) = (1, 5)`: `nums[1] = 0` and `nums[5] = 5`.


=== "Example 2"

    - Input: nums = `[9, 8, 1, 0, 1, 9, 4, 0, 4, 1]`
    - Output: `7`
    - Explanation: The maximum width ramp is achieved at `(i, j) = (2, 9)`: `nums[2] = 1` and `nums[9] = 1`.

## Solution

??? "Key Insight"

    Use a monotonic stack:

    - To maximize the width $j - i$, we want our starting index $i$ to be as far left as possible and its value to be as small as possible.
    - If we look at indices from left to right, we only care about an index if it holds a value smaller than anything we’ve seen so far.
    - If a number is larger than a previous number to its left, it can never serve as a better starting point ($i$) because that previous number is both smaller and further left.
    - Therefore, we can maintain a strictly decreasing stack of indices as our candidates for $i$.

    Calculate Max Width (Right-to-Left):

    - This guarantees we are checking the largest possible widths first.
    - While the stack is not empty and $nums[j] \ge nums[\text{stack.top()}]$, we have found a valid ramp!
    - We calculate the width $j - \text{stack.top()}$ and update our maximum width.
    - We then pop the top index from the stack.

    !!! note

        Why can we safely pop it? Since we are moving $j$ from right to left, the current $j$ is the absolute furthest right element that can pair with that stack top. Any subsequent $j$ we look at will be further to the left, meaning it would only produce a smaller width for that specific starting index.



??? "Full Solution"

    === "Python"

        ```python linenums="1"
        def maxWidthRamp(nums: list[int]) -> int:
        stack = []
        n = len(nums)
        
        # Step 1: Build the monotonic decreasing stack of indices
        for i in range(n):
            if not stack or nums[i] <= nums[stack[-1]]:
                stack.append(i)
                
        max_width = 0
        
        # Step 2: Traverse from right to left to find max width
        for j in range(n - 1, -1, -1):
            while stack and nums[j] >= nums[stack[-1]]:
                max_width = max(max_width, j - stack.pop())
                
        return max_width
        ```

    === "Kotlin"

        ```kotlin linenums="1"
        fun maxWidthRamp(nums: IntArray): Int {
          val stack = ArrayDeque<Int>()
          for ((i, n) in nums.withIndex()) {
            if (stack.isEmpty() || nums[stack.last()] >= n) {
              stack.addLast(i)
            }
          }

          var answer = 0

          for ((i, n) in nums.withIndex().reversed()) {
            while (stack.isNotEmpty() && nums[stack.last()] <= n) {
              answer = max(
                answer,
                i - stack.removeLast()
              )
            }
          }
          return answer
        }
        ```
