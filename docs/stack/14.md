# Buildings Facing Sun

Given a list of building `heights` from east to west, find the the number of buildings that receive sunlight. A building of height $h$ will block sunlight to all buildings after if with heights $<h$.

```kotlin
fun visibleBuildings(heights: IntArray): Int {
  // Fill here
}
```

## Solution

??? "Expand"

    We just need the max height seen from left-to-right. There is no need of a stack.

    ```kotlin
    fun visibleBuildings(heights: IntArray): Int {
      var maxHeight = 0
      var count = 0
      for (h in heights) {
        if (h >= maxHeight) {
          count++
          maxHeight = h
        }
      }
      return count
    }
    ```
